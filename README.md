This is JumbooMovies project

JumbooMovies — это платформа для любителей кино, которая позволяет пользователям оставлять и читать рецензии на фильмы, делиться мнениями и находить новые любимые фильмы. Это приложение создано для того, чтобы объединить киноманов и дать им возможность обмениваться впечатлениями о просмотренных фильмах.

Основные функции:

1) Регистрация и авторизация пользователей:
   
2) Пользователи могут просматривать список фильмов с обложками, жанрами и краткими описаниями.
   
3) Фильтры и сортировка по жанрам, рейтингу, году выпуска и другим критериям.
   
4) Создание и публикация рецензий:

Пользователи могут писать рецензии на фильмы, добавлять свои оценки и теги.
Возможность редактировать или удалять свои рецензии.
Чтение рецензий других пользователей:

5) Возможность просматривать рецензии других пользователей и оставлять комментарии.
   
6) Система оценок рецензий для повышения видимости наиболее полезных отзывов.

Технологии: 
Бэкенд сервис является Spring Boot приложением.
Система сборки - Gradle.
База данных postgresql, для миграции используется Liquibase.
Сервис с базой данных развертывается в docker.
Для работы с бд используется Spring Data Jpa в связке с Hibernate.
Обмен данными с фронтендом происходит по протоколу HTTP в JSON формате, Restful api реализован с помощью Spring Web.
Авторизация производится с помощью Spring Security по протоколу OIDC Connect.
В качестве логгера используется sl4j.
Также присутствует Open Api documentation Swagger UI (http://localhost:8080/swagger-ui/index.html)

Инструкция сетевого взаимодействия фронтенда и бэкенда:
1) Скачаиваем Radmin VPN
2) Создаем сеть на сервере
3) подключаемся к сети на клиенте
4) заменяем localhost на ip-adress сервера в запросах с клиента
